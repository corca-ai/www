# Corca website (`www`)

Marketing site for **Corca** — a static [Astro](https://astro.build) build served
on Cloudflare Workers. Live (beta): https://www.borca.ai

**Start here:** [`docs/index.md`](docs/index.md) is the documentation entry point,
covering architecture, development, deployment, i18n and code quality.

```sh
pnpm install
pnpm dev      # develop locally
pnpm check    # run all quality gates before committing
```
