# Internationalization

The site ships four locales: Korean at the root (`/`), English under `/en/`,
Japanese under `/ja/` and Simplified Chinese under `/zh/`. It uses Astro's
built-in i18n routing with no prefix on the default Korean locale, matching the
original site's URLs.

## Where content lives

- `src/i18n/ui.ts` — the locale list plus navigation, footer and home strings.
- `src/i18n/pageMeta.ts` — per-page SEO titles and descriptions.
- `src/i18n/content/` — news and colleague data, translated per locale.
- `src/i18n/utils.ts` — helpers such as `localizePath` and locale detection.

Each page is authored once as a component and rendered per locale from the four
route trees, as described in [[architecture]]. Translation edits flow into the
build like any other change, so run the checks in [[development]] afterwards.
