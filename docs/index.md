# Corca website documentation

This is the documentation hub for the Corca marketing site (`corca-ai/www`), a
static Astro build served on Cloudflare Workers.

Read the [[architecture]] page for the stack and project layout. Day-to-day work
is covered in [[development]], and shipping is described in [[deployment]].
Translations and page content are explained in [[i18n]], and the linters,
formatters and git hooks live in [[code-quality]]. Common content and
configuration edits are collected as recipes in the [[runbook]].
