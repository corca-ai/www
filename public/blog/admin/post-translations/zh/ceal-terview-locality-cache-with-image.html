主讲人：崔正赫 来源：2026-05-28 AX分享会 主题：局部性·缓存·LLM委托判断

本文并不是止于“写入剪贴板历史记录”的提示集合。正赫在演讲和采访中提出的观点更为广泛。我们刚刚写的命令、刚刚复制的句子、刚刚打开的对话都保留了我们当前工作的流程。与其每次都把它放在抽屉深处，不如把它放在靠近桌子的地方。正赫在CS里说过这句话参考地点哇，用凯茜的感觉解释了。

关键句候选
最近的命令和剪贴板历史记录是我刚刚经历过的想法的密切痕迹。

将本地化视为一种商业工具

希尔问道。

您希望同事在听到您的演讲后立即采取的一项行动是什么？

正赫建议的最具体的行动是每个人都有自己的剪贴板历史记录我们的想法是安装该功能并使用它。工具可以是 Raycast、Alfred 或 Hammerspoon 等选项。在 shell 端，执行历史搜索和先前命令的重写。hstr, 熟悉的ctrl+r相同的习惯属于同一个系列。

这个习惯的价值比“使复制和粘贴更快”更深一些。最近复制的文本和最近输入的命令保留了您刚刚执行的操作的上下文。当前的工作很大程度上与之前的工作相关。既然如此，自然就留下痕迹，以便可以从附近的地方找回，而不是每次都得重新寻找。

简短的判断练习

他们一次一个地出来。目的是练习写作的判断标准，而不是猜测正确答案。

但你不应该缓存所有内容

希尔问道。

您用什么标准来判断演讲原文中“需要刻意把握时机的例外”？

这是写作方向发生重要转变的地方。起初，“为我缓存”这个表达似乎集中在将所有内容放入附近的存储中并快速将其取出。然而，正赫显然例外了。在学习 UI 或掌握心智模型时，有时您必须有意识地手动进行操作。

与正赫的场景是tmux是的。我本可以告诉法学硕士，“像这样分割窗口。”不过，他特意翻看了几遍快捷键列表，手动调整了一下。ctrl+b目的是为了学习入门的感觉。只要有结果，LLM就很快。然而，为了掌握工作空间，有时需要缓慢遍历。

校正对比度
命令历史记录和剪贴板历史记录是“紧密的痕迹”。然而，你在身体和思维中需要学习的不是缓存，而是正确的雕刻。

什么可以委托给LLM，什么不可以委托给LLM

希尔问道。

区分必须有目的地处理而不是留给LLM的任务和可以简单地命令给LLM的任务的标准是什么？

正赫的标准很简单。如果这是你已经擅长的事情或者你不打算立即自己做的事情，你可以要求LLM去做。另一方面，需要掌握的东西、需要培养肌肉记忆的东西、需要学习理解的东西，都是自己刻意一步步做的，甚至要求LLM“帮我做”。

该标准并不是为了减少 LLM 的使用。相反，它更接近更好授权的标准。要知道什么可以留给外部工具，你必须知道什么应该留在你自己内部。正赫提到的“雕刻”这个词在这里很重要。缓存是可以快速访问和使用的就近存储。铭刻是慢慢地、反复地放入身体和思想的基本层中的东西。

正赫主要使用“雕刻”这个词来表示熟悉的重复。然而，重复并不以简单的机械训练结束。随着重复处理碎片的流畅度的提高，最终会带来更高层次的理解。换句话说，记住快捷方式的行为会在某些时候改变您查看工作区的方式。

安静的检查站

想一想今天我想委托给我的法学硕士的一项任务。这项工作是“我已经知道的东西，或者不需要立即制作成我自己的东西”，还是“我需要雕刻到自己手中的东西，即使它很慢”？

作为这篇文章的标准，我先看“是否应该留在我体内”，然后再看速度。如果你只需要结果并且对你的下一个决定没有太大影响，那么委托是可以的。另一方面，如果它是你将来会经常处理的基本操作、思维方式和工具的心智模型，那么从长远来看，至少自己过一遍会带来更大的流畅性。

仅通过授权技巧无法实现生产力。

小麦打结了

这篇文章的结论是更接近于“区分什么该托付给LLM，什么该刻自己”，还是更接近“故意低效地重复需要刻的东西”？

正赫要求采访者以理智的方式结束这一部分。因此，本文不是仅选择两者之一，而是将这两个句子连接起来。

人工智能时代的生产力不是来自于将一切委托给别人的能力，而是来自于选择不委托什么的意识。

不授权并不意味着拒绝工具。是否攻读LLM并不是禁欲的问题。标准是：“这个能力是可以放在我的体外，还是应该刻在我的身体和思想里？”你可以把你已经擅长的或者你现在不需要自己做的LLM工作留给你。但是tmux的ctrl+b就像，当谈到创造我的工作空间感时，暂时采取缓慢的路径实际上会在以后走上更快的路径。

Locality 和 Cathy 告诉我们“不要忘记过去的想法”。至于正赫的例外，这里补充一句。 “该靠近的就靠近，该刻的不要太快。”

您可以立即尝试的一个小实验

  • 打开剪贴板历史记录工具，全天记录您检索和重写“刚刚复制的内容”的时刻。
  • 有意识地使用shell命令历史搜索，减少重复输入的命令数量。
  • 在询问 LLM 之前，请停顿 10 秒并询问。 “我只需要一个结果，还是应该把它刻在手上？”
